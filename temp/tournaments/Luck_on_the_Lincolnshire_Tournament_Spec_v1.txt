6) LINCOLNSHIRE-STYLE MODIFIED SCORING (STANDARD BOARDS)
====================================================================
Purpose:
- Simulate the ‚Äúmissing triples / altered bull‚Äù Lincolnshire vibe without altering boards physically.

SCORING RULES (LOCKED)
1) Triples score as Singles
- Any dart landing in a triple segment scores the SINGLE value of that number.
  Example: T20 counts as 20 (NOT 60)

2) Doubles score normally
- Doubles remain standard (D20 = 40, etc.)

3) Bull values adjusted
- Inner bull (bullseye) = 50
- Outer bull = 10

4) Singles and misses are standard
- Singles score face value
- Miss / bounce-out = 0

Why this works:
- Removes ‚Äúpower scoring‚Äù without confusing tape jobs
- Keeps games closer and more strategic
- Makes bull useful but not dominant

Posting requirement:
- Print a simple rules card for EVERY board:
  ‚ÄúTriples = Singles ‚Ä¢ Bull 50/10 ‚Ä¢ 401 DIDO ‚Ä¢ BO3 (Final BO5)‚Äù
